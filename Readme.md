### A curated collection of university projects demonstrating strong foundations in statistical modeling, data analysis, and applied machine learning. All projects received the highest academic distinction.

Multiple Linear Regression (R) – Statistical Analysis of European Countries
Built and evaluated regression models in R to study economic and social indicators across European countries. This project focused on feature selection, multicollinearity, model diagnostics, and statistical inference, with emphasis on producing interpretable, high-quality predictive models.

Macroeconomic Spatial Analysis (GeoDa) – MacroStatistics Project
Conducted spatial econometric analysis of macroeconomic indicators using GeoDa, identifying regional dependencies and spatial autocorrelation patterns.
This project was selected and presented at the Scientific Communications for Students conference, recognizing its methodological rigor and analytical contribution.

Together, these projects highlight hands-on experience in:

Statistical modeling & regression

Data cleaning, exploration, and hypothesis testing

Spatial and macroeconomic analysis

Translating mathematical models into real-world insights

They form a strong foundation for work in machine learning engineering, data science, and quantitative modeling.
