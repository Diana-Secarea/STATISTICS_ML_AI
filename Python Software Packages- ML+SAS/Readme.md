### A curated collection of university projects demonstrating strong foundations in statistical modeling, data analysis, and applied machine learning. All projects received the highest academic distinction.

Linear Discriminant Analysis (Python) – LDA Analysis & Predictions
Applied LDA for supervised classification of country income groups using World Bank indicators (GDP per capita, unemployment, inflation, life expectancy, school enrollment). Demonstrated rigorous methodology: formal assumption checks (multivariate normality via Shapiro–Wilk and KDE, homogeneity of variance, multicollinearity via VIF), data merging from multiple sources, standardization, and a reproducible pipeline. Delivered full model evaluation (classification report, raw and normalized confusion matrices), interpretable discriminant coefficients and feature importance, and visual diagnostics (LDA scatter plots, coefficient heatmaps, correlation heatmaps)—showcasing best practices in interpretable classification and assumption-driven modeling.



BERT for Named Entity Recognition – Personal Data Anonymization
Explored transformer-based deep learning for privacy-preserving NLP: applied a BERT model for named entity recognition (NER) to support personal data anonymization. Demonstrates familiarity with state-of-the-art language models, entity extraction, and the intersection of machine learning with data privacy and compliance.


Multiple Linear Regression (R) – Statistical Analysis of European Countries
Built and evaluated regression models in R to study economic and social indicators across European countries. This project focused on feature selection, multicollinearity, model diagnostics, and statistical inference, with emphasis on producing interpretable, high-quality predictive models.
